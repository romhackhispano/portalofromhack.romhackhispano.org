Utilidades

  
  
Ultilidades usadas por la mayoría de los ROM Hackers
 
Editores Hexadecimales:
Translhextion 1.6c
Maravilloso editor hexadecimal en el cual puedes,
extraer e insertar scripts, cargar tablas Thingy, etc.
Un buen editor hexadecimal.
ThingyV
Modificación del editor hexadecimal Thingy realizada por
Vegetal Gibber, una característica que se ha de destacar
es la posibilidad de cargar ISOS.
Editores Gráficos:
Tile Layer Pro
Este editor sirve para modificar o editar los tiles de
los diferentes tipos de ROM, sea GB, SNES, NES,
SEGA GENESIS entre otros.
Tile Molester
Es un editor programado en Java.
Necesitas tener instalada la Maquina Virtual de Java
Búsqueda Relativa:
SearchRX
Es un programa que hará una búsqueda relativa en tu rom
el cual te ayudara a la hora de hacer tu tabla Thingy.
Esencial.
SearchR2
Este es un programa en DOS si es que no me equivoco.
Donde también puedes hacer una Búsqueda Relativa
Tablas:
Tabular
Fácil programa donde podrás completar tu tabla Thingy.
Parches:
SNEStool
Sino me equivoco hay varias versiones.
Con este programita podrás aplicar y crear parches,
también tiene más funciones.
